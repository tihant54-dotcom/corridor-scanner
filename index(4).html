⚡ Как это работает
Запросы к API Fonbet и Maxline выполняются прямо из вашего браузера через Telegram Mini App. Никаких серверов — ваш IP, никаких блокировок.

Парсер ищет совпадающие матчи между БК и сравнивает котировки по тоталам.
📐 Формула коридора
margin = 1/k1 + 1/k2

Если margin < 1 → коридор существует.

profit% = (1/margin − 1) × 100

Оптимальные ставки при банке B:
S1 = B / (1 + k2/k1)
S2 = B − S1
